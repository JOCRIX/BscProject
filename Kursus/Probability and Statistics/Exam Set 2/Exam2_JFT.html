To find out which quarters differ, we can make pairwise comparisons. We could use the pairwise t-tests as in Part I, Exercise 3. However, there is a problem with this approach:

  1. How many pairwise comparisons of quarterly mean values can we make? What is the probability of rejecting a single true null-hypothesis? If we test several true null-hypotheses, explain why the probability of falsely rejecting at least one of them is larger than the significance level.

We can make 6 different caparisons between means. This can be found using the binomial coefficient: We have 4 elements (seasons) and two comparisons such that we get: \(\binom{n}{r} = \frac{n!}{r!(n-r)!} \Rightarrow \binom{4}{2} = \frac{4!}{2!(2)!} = 6\) We can make 6 different comparisons, given a probability of error of 5% for each the final probability of making at least one error is 6*0.05 = 0.3 or 30%. If we do not care what comparison is at “fault” or erroneous, but only care if we make an error, then there is 5% chance at each comparison for an error, and this adds up to a total chance for an error to be 30%. We can say that the event for each comparison is mutually exclusive. This is also the reason why we can add their probabilities together. We can say that the sample space contains 6 independent events, each event is a comparison either being true or false.

The probability of rejecting a single true null-hyp is 5% given a level of significance of \(\alpha = 0.05\).

This phenomenon is known as multiple testing. We can adjust the pairwise t-tests for multiple testing by performing a so-called Tukey’s test. It replaces the t-distribution by a different distribution (the studentized range distribution) that takes multiple testing into account.
